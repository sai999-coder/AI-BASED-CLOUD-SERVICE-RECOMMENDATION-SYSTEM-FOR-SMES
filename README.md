
# AI-BASED CLOUD SERVICE RECOMMENDATION SYSTEM FOR SMES

This project develops an AI-powered recommendation engine that helps Small and Medium Enterprises (SMEs) select optimal cloud services. By analyzing business needs like workload type, budget constraints, scalability requirements, and compliance standards, the system suggests tailored providers such as AWS, Azure, Google Cloud, or others. Built with machine learning models, it simplifies cloud adoption for SMEs lacking dedicated IT teams.


## Features
Personalized Recommendations: Matches services to SME profiles using collaborative filtering and content-based algorithms.

Cost Optimization: Compares pricing models (pay-as-you-go, reserved instances) with predictive analytics for total cost of ownership.

Multi-Provider Support: Evaluates AWS, Azure, GCP, DigitalOcean, and more across 50+ metrics.

User-Friendly Interface: Web dashboard for inputting requirements and viewing ranked suggestions with explanations.

Performance Metrics: Achieves 92% accuracy on test datasets; processes recommendations in under 5 seconds.
